📁 /src/
│
├── main.cpp              → App entry point, setup loop, task scheduler
│
├── ui/
│   └── display_ui.cpp    → OLED screen views, rotary UI, page routing
│   └── encoder.cpp       → Rotary encoder input handler
│
├── sensors/
│   └── thermocouple.cpp  → MAX6675 reading, smoothing, error handling
│   └── eeprom.cpp        → Preferences (target temp, last state)
│
├── control/
│   └── fan_control.cpp   → PWM fan control via MOSFET
│   └── servo_control.cpp → Damper servo logic
│   └── pid.cpp           → PID logic (optional for temp control)
│
├── network/
│   └── wifi.cpp          → WiFiManager setup, reconnect watchdog
│   └── web_server.cpp    → Async web server routes and API
│   └── ota.cpp           → OTA update support (via ElegantOTA)
│
├── utils/
│   └── logger.cpp        → Serial logging wrapper
│   └── scheduler.cpp     → Task ticker/timer manager (e.g. Ticker or millis-based)
